# 💈 Sistema de Barbearia

Sistema simples de gerenciamento de barbearia desenvolvido em Java utilizando programação orientada a objetos e padrão DAO.

---

## Tecnologias Utilizadas

- ☕ Java
- 🖥️ Swing
- 🗄️ MySQL
- 🔌 JDBC
- 📦 Maven
